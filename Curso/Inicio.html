Primeiro Modulo

Nesta primeira parte, estou colocando em pratica os primeiros ensinamentos.

Para começar, foi apresentado algumas profissoes que podem ser seguidas e suas determinadas funções.

  1. Frontend : Pessoas que trabalham na parte da frente do site. A parte visual
  2. Backend : Pessoas que trabalham com a construção do site, basicamente a formação do "esqueleto" de um site.
  3. Mobile: Pessoas que trabalham com a criaçãode aplicativos, aqui foi explicado que é mais a parte da Inter face do aplicativo.

Recebi uma dica importante, começar os estudos na seguinte ordem

  • Html 5
  • CSS 3
  • Java Script

Para a continuação e avanço na carreira

  • Bootstrap
  • React, Vue ou Angular

Deixei a ultima linha em negrito porque são 3 opçoes, e você deve escolher somente uma.

Algumas definições

Html

O Html é responsavel pela criação dos codigos de formação do site. Por exemplo; Agora você consegue vizualizar tudo o que eu estou escrevendo por conta do HTML.

Css

O Css é responsavel por toda a personalização do site, por enquanto o CSS não está sendo utilizado, mas, em breve você podera notar coisas que há algumas coisas decorativas no site.

No primeiro modulo eu também aprendi sobre onde criar o meu codigo, tags, atributos e sobre a estrutura basica de um site

Segundo modulo
